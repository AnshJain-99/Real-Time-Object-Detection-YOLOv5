# Real-Time Object Detection System

A real-time object detection system developed using YOLO and OpenCV for detecting and classifying multiple objects from video streams.

## Features
- Real-time object detection
- Multi-object classification
- Video stream processing
- Bounding box visualization
- High-speed inference

## Technologies Used
- Python
- YOLO
- OpenCV
- PyTorch

## Run

pip install -r requirements.txt

python detect.py
